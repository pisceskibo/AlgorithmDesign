1) T(1) = 1, and for all n ≥ 2, T(n) = 3T(n − 1) + 2
T(n) = 3T(n-1) + 2*1
     = 3(3T(n-2) + 2) + 2 = 3^2 T(n-2) + 2*(3 + 1)
     = 3^3 T(n-3) + 2*(3^2 + 3^1 + 3^0)
     = ...
     = 3^{n-1} T(1) + 2*(3^{n-2} + 3^{n-3} + ... + 3^1 + 3^0)
     = 3^{n-1} + 2*(1-3^{n-1})/(1 - 3)
     = 3^{n-1} + 3^{n-1} - 1
     = 2 * 3^{n-1} - 1  

2) T(1) = 3, and for all n ≥ 2, T(n) = T(n − 1) + 2n − 3
T(n) = T(n-1) + 2*1(n-1) - 1
     = T(n-2) + 2*(n-2) - 1 + 2*(n-2) - 1 = T(n-2) + 2*2(n-2) - 2
     = T(n-3) + 2*(n-3) - 1 + 2*(n-3) - 1 + 2*(n-3) - 1 = T(n-3) + 2*3(n-3) - 3 
     = ...
     = T(n - (n - 1)) + 2*(n-1)(n-(n-1)) - (n-1)
     = T(1) + 2(n-1) - (n-1) = T(1) + n - 1 = 3 + n - 1 
     = n + 2

3) T(1) = 1, and for all n ≥ 2, T(n) = 2T(n − 1) + n − 1
T(n) = 2T(n-1) + 1(n-1)
     = 2^2T(n-2) + 2(n-2) + (n-2) = 2^2T(n-2) + 3(n-2) = 2^2T(n-2) + (2*2-1)(n-2)
     = 2^3T(n-3) + 5(n-3) = 2^3T(n-3) + (2*3 - 1)(n-3)
     = ...
     = 2^{n-1}T(n-(n-1)) + (2*(n-1) - 1)(n-(n-1))
     = 2^{n-1}T(1) + 2n - 3 = 2^{n-1} + 2n - 3

4) T(1) = 5, and for all n ≥ 2, T(n) = 2T(n − 1) + 3n + 1
T(n) = 2T(n-1) + 3(n-1) + 4(1)
     = 2(2T(n-2) + 3(n-2) + 4) + 3(n-2) + 4 = 2^2T(n-2) + (2^1*3+3)(n-2) + 4(2 + 1)
     = 2^2(2T(n-3) + 3(n-3) + 4) + (2^1*3+3)(n-3) + 4(2 + 1) = 2^3T(n-3) + (2^2*3+2^1*3+3)(n-3) + 4(2^2 + 2^1 + 2^0)
     = ...
     = 2^{n-1}T(n-(n-1)) + (2^{n-2}*3 + 2^{n-3}*3 + ... + 2^1*3 + 2^0*3) + 4(2^{n-1} + 2^{n-2} + ... + 2^1 + 2^0)
     = 2^{n-1}T(1) + 3*(1 - 2^{n-1})/(1 - 2) + 4*(1 - 2^n)/(1-2)
     = 5*2^{n-1} + 3*(2^{n-1} - 1) + 4*(2^n - 1)
     = 5*2^{n-1} + 3*2^{n-1} - 3 + 2*2^{n-1} - 4
     = 10*2^{n-1} - 7 = 5*2^n - 7
